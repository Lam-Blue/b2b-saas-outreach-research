## Fallback Rules

- If a public LinkedIn post is inaccessible, create `research/linkedin-posts/[expert-slug]/availability.md` and use the fallback source type listed above.
- If a queued transcript fails, record the exact failure in `collection-status.md` and `collection-audit.md`, then use a pre-approved fallback source for the same area.
- Do not replace unavailable material with unrelated recent content.
