Source Capture
Purpose

This rule defines how public material is captured, stored, and referenced.

The repository should preserve enough evidence to verify conclusions without becoming a copy of the internet.

File destinations

Use these folders:

research/linkedin-posts/
research/youtube-transcripts/
research/other/

Use this naming format:

[candidate-slug]--[source-slug].md

Examples:

jason-bay--cold-call-objection-framework.md
will-allred--email-benchmark-analysis.md

Do not create new folders unless the task plan requires them.

## Why this matters

One or two sentences explaining which candidate criterion or research area this source supports.

## Limits or uncertainty

State missing date, incomplete access, unsupported claims, or any limitation that affects confidence.
Date rules
Use the original publication date whenever it is visible.
Record the access date for every source.
If no publication date is visible, write Unknown.
Never infer recency from comments, page design, search snippets, or follower activity.
Follow the recency threshold defined in research/sources.md or the active task plan.
Platform restrictions

Do not bypass logins, paywalls, CAPTCHAs, robots controls, or platform restrictions.

If a source cannot be accessed reliably:

record the public URL if available
mark the source as inaccessible or partially accessible
do not reconstruct its content from a search snippet
find a different verifiable source where possible
Copyright-safe capture

Do not commit full third-party video transcripts, full copied LinkedIn posts, or bulk copied content to a public repository.

For a video or podcast:

capture the canonical source URL
capture only the relevant timestamped passages
write a concise original summary
retain enough context to support the conclusion

A transcript API may be used as a navigation aid, but not as a reason to commit an entire third-party transcript.

Quality standard

A source is useful only when it contributes at least one of these:

direct evidence of practice
evidence of a specific operating area
a concrete, reusable outbound insight
credible evidence of recent public material

Discard generic or duplicate material rather than saving it “just in case.”

A copyright year, page footer year, access date, search-engine snippet, or apparent current branding is not a publication date.

It may provide weak currency context, but it must never satisfy a requirement for dated original material or recency.
