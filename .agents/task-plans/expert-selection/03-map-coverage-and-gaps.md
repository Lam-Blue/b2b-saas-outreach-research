# Task 03 — Map Current Coverage and Define Discovery Gaps

## Purpose

Review the completed initial-candidate research as a group.

Identify:

* which project areas already have strong candidates
* which areas have weak, overlapping, or unverified coverage
* where an external candidate could be meaningfully better than the initial list

This is an analysis task only.

Do not perform new web research.

## Required skill

Use the expert-screening skill.

## Inputs

* `research/sources.md`
* `research/other/initial-landscape-overview.md`
* `output-by-AI/research-log.md`
* all initial candidate dossiers
* `output-by-AI/source-library.md`
* repository rules
* workflow README
* this task file

## Required output

Create:

```text
output-by-AI/discovery/coverage-gap-brief.md
```

Use this structure:

```md
# Coverage Gap Brief

## Current coverage map

For each area defined in `research/sources.md`:

- candidates currently mapped to this area
- evidence strength
- source IDs supporting the mapping
- overlap with other candidates
- material availability for later retrieval

## Strong coverage

List areas where at least one candidate has clear practitioner evidence and usable public material.

## Weak or uncertain coverage

List areas where:

- practitioner proof is insufficient
- the available candidate is too broad or generic
- recent usable material is weak
- multiple candidates overlap heavily
- no candidate clearly fits the area

## External discovery briefs

For each gap or weak area, define:

- target area
- why the current coverage is insufficient
- the kind of practitioner needed
- required proof of practice
- preferred public material
- characteristics to avoid
- whether the goal is gap coverage or a stronger replacement candidate
```

## Discovery design rules

External discovery must be driven by two needs:

1. **Coverage gaps**
   Find candidates for areas with no sufficiently strong fit.

2. **Challenge searches**
   Find stronger alternatives when an existing candidate is only borderline, weakly evidenced, or overlapping with another candidate.

Do not recommend external discovery merely to make the list larger.

## Completion condition

This task is complete only when:

* every area defined in `research/sources.md` is evaluated using existing evidence
* every discovery brief is tied to a specific observed gap or weakness
* no new sources or candidates are added
* `research/sources.md` remains unchanged
